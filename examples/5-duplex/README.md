# Duplex Streaming Example

Demonstrates bidirectional duplex streaming over HTTP for interactive applications.

## Run

```bash
# Terminal 1: Start server
bun run examples/5-duplex/server.ts

# Terminal 2: Run client
bun run examples/5-duplex/client.ts
```

## What it demonstrates

- Bidirectional duplex streaming over HTTP
- Interactive server-client communication
- Server calling client actions during streaming
- Real-time collaborative features
- Using startDuplex for bidirectional connections
